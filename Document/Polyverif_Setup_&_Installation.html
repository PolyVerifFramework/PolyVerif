PolyVerification Suite Setup and Configuration

This document contains the information for setting up and installing the PolyVerification suite on Linux/Unix. We have modified some open-source packages which are added in the repository.

System Requirements:

  • CPU: at least 8 core CPU
  • GPU: Nvidia GTX 1080 (8GB memory) or higher
  • OS: Ubuntu 18.04 or 20.04 (20.04 is recommended)
  • Python 3.8

1. Setup Docker

For Docker installation, please refer to Docker_Installation.html found in the Documents directory. Alternatively, you can visit the PolyVerif GitHub repository.

2. PolyVerif Setup

For PolyVerif setup, run the below commands. It will automatically set up and install all packages of the PolyVerif suite.

$ docker pull polyverif/ossdc-sim:v1.1 && docker run -it -e HOST_USERNAME=$USER -v $HOME:/host-home polyverif/ossdc-sim:v1.1
        
$ ./adehome/build_env.sh
